Automation Testing for Demo App

Overview
This project demonstrates the automation of the Login Page and Home Page of a demo web application using Selenium WebDriver and TestNG framework. It covers the following functionalities:

1. Login Page Automation: Test all possible scenarios for the login functionality.
2. Home Page Automation: Test the sorting functionality on the home page table.
(Detailed information on test cases is covered in the excel file)

Technology Stack
- Programming Language: Java
- Test Automation Framework: TestNG
- Web Automation Tool: Selenium WebDriver
- Browser: Google Chrome (with ChromeDriver)
- Data-Driven Testing: Properties file for login credentials
- Reporting: HTML Report via TestNG

Run the TestNG Tests via IDE (Eclipse)
- Right-click on the testng.xml file and 'Run as' -> 'TestNG Suite'.
